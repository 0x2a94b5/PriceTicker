# 安全策略 / Security Policy

## 支持范围

当前仅维护最新发布版本和 `main` 分支。旧版本发现的问题通常会在后续版本中修复，不单独回补。

| 版本 | 安全更新 |
| --- | --- |
| 最新 Release | 支持 |
| `main` | 支持 |
| 更早版本 | 不支持 |

## 私下报告漏洞

请勿通过公开 Issue 披露漏洞、利用代码、凭据或用户数据。优先使用 GitHub 的 [Private vulnerability reporting](https://github.com/0x2a94b5/PriceTicker/security/advisories/new) 提交报告。

报告建议包含：

- 受影响版本或提交
- 复现步骤和影响分析
- 概念验证或日志（请先移除敏感数据）
- 可行的修复建议（如有）

维护者会尽力在 7 天内确认收到报告，并在完成评估后同步处理计划。修复发布前，请不要公开漏洞细节。

如果 Private vulnerability reporting 尚未启用，请通过维护者 [GitHub 个人资料](https://github.com/0x2a94b5) 中列出的私密联系方式报告。

## English summary

Only the latest release and `main` are supported. Do not disclose vulnerabilities in public issues. Use [GitHub private vulnerability reporting](https://github.com/0x2a94b5/PriceTicker/security/advisories/new), including affected versions, reproduction steps, impact, and a sanitized proof of concept. Please allow maintainers time to assess and fix the issue before public disclosure.
