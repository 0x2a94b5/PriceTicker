# 获取帮助 / Support

## 使用问题

提交问题前，请先查看 [README](README.md)、[现有 Issues](https://github.com/0x2a94b5/PriceTicker/issues) 和 [Releases](https://github.com/0x2a94b5/PriceTicker/releases)。

- 可稳定复现的应用故障：使用 Bug Report 模板。
- 新功能或体验建议：使用 Feature Request 模板。
- Binance API 服务状态、账户或交易问题：请联系 Binance；本项目无法处理第三方账户问题。
- 安全漏洞：按照 [SECURITY.md](SECURITY.md) 私下报告，不要创建公开 Issue。

维护者会根据影响范围和可复现程度安排处理，但不承诺响应时间或商业级支持。

## English summary

Use GitHub Issues for reproducible bugs and feature requests after checking the README and existing issues. Contact Binance for API availability or account-related matters. Report security vulnerabilities privately according to [SECURITY.md](SECURITY.md). Community support is best-effort.
